# MakersFlow Mobile App Assets

This directory contains the asset files used across the MakersFlow (Edodwaja) React Native application, structured for clean and scalable asset management.

# logos

Application logo, app icon, brand marks.

# splash

Splash screen images.

# onboarding

Images used in onboarding screens.

# placeholders

Fallback images when remote content is unavailable.

# certificates

Certificate decorations such as ribbons, seals, watermarks, backgrounds, and signature images.

# partners

Partner logos such as Flow Bus and Edodwaja Foundation.

# courses

Static course images used when no remote thumbnail exists.

# products

Fallback product images.

# banners

Promotional banners shown on the Home screen.

# avatars

Default user avatars.

# empty-states

Illustrations shown for empty screens such as "No Courses", "No Orders", and "No Internet".

# illustrations

General illustrations used across the application.

# icons

Custom PNG/SVG icons grouped by feature.

# fonts

Custom font files.

# animations

Lottie JSON animation files.
